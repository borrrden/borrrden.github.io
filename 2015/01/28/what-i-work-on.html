What I Work On

28 Jan 2015

For my first entry I would like to explain in my own words what it is exactly that I work on. I work on a product named Couchbase Lite, which is a component of Couchbase Mobile. Couchbase Mobile is a solution that allows a developer to store data using the NoSQL paradigm locally on mobile devices, with an easy way to sync and replicate data to servers via something called Sync Gateway.

The benefit to this is that the full persistence and replication stack is already taken care of for you. You can view the sources of the various implementations (iOS, Android, .NET, etc) via the Couchbase GitHub page. Specifically I am in charge of implementing the .NET version. Later I may write some articles exploring the internals of Couchbase Lite, so stay tuned!
